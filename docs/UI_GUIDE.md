# RemindMe UI Guide

RemindMe는 사용자가 물어본 영어 표현과 직접 추가한 표현을 기억해두고, 사용자가 정한 시간에 짧은 퀴즈로 다시 꺼내주는 복습 서비스다.

이 UI의 핵심은 `검색처럼 가볍게 입력하고, 알림처럼 부담 없이 복습한다`는 점이다. 사용자가 공부 앱을 새로 배운다는 느낌보다, 평소 AI에게 영어를 물어보던 흐름 안에서 자연스럽게 복습 루틴이 생기는 경험을 목표로 한다.

## Product Goal

- 사용자가 질문한 영어 단어, 숙어, 문장 표현을 자동으로 복습 대상으로 저장한다.
- 이미 알고 있는 표현도 직접 추가할 수 있게 한다.
- 뜻, 예문, 퀴즈 보기는 AI가 생성해서 사용자의 입력 부담을 줄인다.
- 알림별로 시간과 복습량을 설정해 출근길, 점심시간, 퇴근길처럼 실제 생활 루틴에 붙인다.
- 모든 표현을 벡터 검색에만 맡기지 않고, 날짜, 사용자, 쿼리, 응답, 리마인드 회수, 정답률 같은 명시적 데이터를 함께 사용해 할루시네이션을 줄인다.

## UX Principles

### 1. Home Is Only For Asking

홈 화면은 검색창 하나에 집중한다.

- 홈에서는 영어 표현을 물어보거나 직접 추가로 들어가는 것만 제공한다.
- 홈에 카드, 통계, 복습량 설정, 예문 버튼, 숙어 버튼을 늘어놓지 않는다.
- 사용자는 앱을 켜자마자 `무엇을 물어볼까?`만 생각하면 된다.

### 2. Manual Add Is Minimal

직접 추가는 사용자가 이미 아는 표현을 복습 대상에 넣는 기능이다.

- 입력값은 표현 하나만 받는다.
- 단어, 숙어, 문장패턴 같은 타입 선택을 요구하지 않는다.
- 뜻과 예문은 사용자가 작성하지 않는다.
- AI가 뜻, 예문, 퀴즈 선택지를 생성한다는 안내만 제공한다.
- 직접 추가 화면에는 홈으로 돌아가는 명확한 뒤로가기 링크가 있어야 한다.

### 3. Reminder Settings Are Per Alarm

복습량은 하루 전체 기준이 아니라 알림별로 설정한다.

- `08:10 · 10분 · 5문제`처럼 시간과 복습량이 한 세트로 보여야 한다.
- `출근길`, `점심` 같은 라벨은 UI에서 기본 제공하지 않는다.
- 사용자는 `+` 버튼을 눌러 시간과 복습량을 직접 고른 뒤 알림을 추가한다.
- `+` 버튼을 누르는 즉시 알림이 자동 생성되면 안 된다.

### 4. Sidebar Is Navigation, Not A Dumping Ground

사이드바는 주요 영역으로 이동하는 메뉴다.

현재 메뉴는 다음 네 가지로 유지한다.

- 홈
- 바로 복습
- 시간 설정
- 기억 저장소

사이드바에는 직접 추가 메뉴를 별도로 만들지 않는다. 직접 추가는 홈의 검색창 액션으로만 진입한다.

### 5. Quiz Should Feel Short And Predictable

사용자는 알림을 눌렀을 때 긴 학습 세션에 끌려가는 느낌을 받으면 안 된다.

- 퀴즈 수와 예상 시간을 먼저 보여준다.
- 한 문제 카드에는 하나의 질문만 담는다.
- 정답 피드백은 간결하게 제공한다.
- 예문은 정답 이해를 돕는 용도로만 사용한다.

## Visual Direction

현재 디자인 방향은 `violet gradient search companion`이다.

- 전체 분위기는 바이올렛 계열이다.
- 배경은 정적인 단색이 아니라 은은하게 흐르는 그라데이션을 사용한다.
- 홈 검색창은 Gemini 검색창처럼 크고 부드러운 프롬프트 카드 느낌을 가진다.
- 사이드바와 주요 카드는 반투명 글래스 느낌을 사용하되, 텍스트 가독성을 해치지 않는다.
- Duolingo식 초록색 게임 UI로 바꾸지 않는다.
- 인터페이스 구조를 크게 바꾸기보다 현재 구조 위에 스타일을 다듬는다.

## Typography

- 기본 폰트는 `IBM Plex Sans KR`를 사용한다.
- 브랜드 마크는 `Newsreader`를 사용한다.
- 버튼, 메뉴, 카드 타이틀은 굵고 짧게 쓴다.
- 설명 문장은 길게 늘어놓지 않는다.
- 영어 표현은 카드 안에서 한눈에 보이도록 굵게 표시한다.

## Layout

### Home

홈 화면 구성은 다음을 유지한다.

- 중앙 정렬된 검색 프롬프트 카드
- 텍스트 입력 영역
- `+ 직접 추가` 액션
- 원형 전송 버튼

홈 화면에 두지 말아야 할 것:

- 복습 통계
- 알림 설정
- 예문 / 숙어 바로가기 버튼
- 기억 저장소 리스트
- 과도한 설명 문구

### Sidebar

사이드바는 메뉴 버튼으로 열고 닫는다.

- 닫힌 상태에서는 화면 왼쪽의 `Menu` 버튼만 보인다.
- 열린 상태에서는 뒤 화면이 fade out 처리되어야 한다.
- 배경은 흐려지지만 사이드바 자체는 선명해야 한다.
- 메뉴 선택 시 해당 패널로 이동하고 사이드바는 닫힌다.

### Routine

시간 설정 화면은 알림 관리에만 집중한다.

- 상단에는 현재 알림 수와 선택된 시간을 보여준다.
- 알림은 bar 형태 카드로 보여준다.
- 각 알림 카드에는 시간, 예상 시간, 문제 수만 표시한다.
- 추가 폼에서는 시간과 복습량을 함께 선택한다.

기본 복습량 옵션:

- 10분 · 5문제
- 30분 · 15문제
- 60분 · 30문제

### Manual Add

직접 추가 화면은 다음 정보만 받는다.

- 표현 입력 필드
- AI가 뜻과 예문을 생성한다는 안내
- 추가 버튼

제거해야 할 것:

- 뜻 입력 필드
- 예문 입력 필드
- 단어 / 숙어 / 문장패턴 선택
- `AI로 정리해서 넣기` 같은 무거운 버튼 문구

### Memory

기억 저장소는 사용자가 물어봤거나 직접 추가한 표현의 상태를 보여준다.

표현 상태 예시:

- 다시 볼 표현
- 오늘 추가
- 안정권
- 직접 추가

카드에는 표현, 짧은 설명, 정답률 또는 복습 상태를 표시한다.

## Interaction Rules

- `+ 직접 추가`를 누르면 manual 패널로 이동한다.
- manual 패널의 `홈으로`를 누르면 home 패널로 돌아간다.
- `+` 알림 버튼은 추가 폼을 여는 역할만 한다.
- 알림 추가 폼에서 `추가`를 눌렀을 때만 새 알림이 생성된다.
- 알림 카드를 클릭하면 해당 알림이 선택 상태가 된다.
- 퀴즈 선택지를 누르면 정답 피드백이 표시된다.
- 사이드바 backdrop을 누르면 사이드바가 닫힌다.

## Content Tone

문구는 가볍고 명확해야 한다.

좋은 문구:

- 영어 표현, 숙어, 예문을 물어보세요
- 이미 아는 표현도 넣어두세요.
- 뜻과 예문은 AI가 자동으로 생성해서 복습 카드에 붙여둘게요.
- 10분 · 5문제

피해야 할 문구:

- AI 기반 개인화 지식 관리 시스템
- 학습 데이터를 기반으로 최적화된 리마인드 경험을 제공합니다
- Expression Intelligence Dashboard
- 출근길 / 점심 / 퇴근길 같은 고정 라벨

## Accessibility

- 메뉴 버튼은 `aria-expanded` 상태를 갱신해야 한다.
- 검색 입력과 직접 추가 입력에는 명확한 `aria-label`이 있어야 한다.
- 사이드바 backdrop은 시각 효과용이므로 `aria-hidden="true"`를 유지한다.
- 버튼은 색상만으로 상태를 구분하지 않고 active 스타일과 텍스트를 함께 사용한다.
- 모바일에서도 주요 액션이 한 손 조작 가능한 위치에 있어야 한다.

## Responsive Behavior

- 모바일에서는 사이드바가 기본적으로 닫힌다.
- 콘텐츠 영역은 화면 폭에 맞춰 중앙 정렬한다.
- 홈 검색창은 너무 넓게 늘어나지 않도록 최대 폭을 제한한다.
- 시간 설정의 알림 카드는 세로로 쌓여도 정보가 잘려서는 안 된다.
- 터치 대상은 충분한 높이를 유지한다.

## Do / Don't

Do:

- 홈은 검색창 중심으로 비워둔다.
- 사용자가 직접 입력해야 하는 정보를 줄인다.
- 알림별 복습량을 명확하게 보여준다.
- 사이드바는 필요한 메뉴만 둔다.
- 바이올렛 그라데이션과 글래스 카드 톤을 유지한다.

Don't:

- 홈에 기능을 계속 추가하지 않는다.
- 직접 추가에서 뜻, 예문, 타입을 묻지 않는다.
- `+` 버튼 클릭만으로 알림을 만들지 않는다.
- 사이드 메뉴에 직접 추가를 다시 넣지 않는다.
- Duolingo 스타일처럼 전체 인터페이스를 게임맵으로 바꾸지 않는다.
